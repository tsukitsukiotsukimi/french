# French Study Webpage

このフォルダは、3つのMarkdown教材を統合したオフライン学習用Webページです。

## 開き方

1. `index.html` をブラウザで開く。
2. 検索・章ジャンプ・学習チェックを利用する。
3. 元Markdownは `source_markdown/` に同梱しています。

## ファイル構成

- `index.html`: 統合Web教材
- `styles.css`: デザイン
- `script.js`: 検索・学習チェック保存
- `source_markdown/`: 元Markdown本文（内容保持用）
- `manifest.json`: 元ファイルのハッシュと統計
